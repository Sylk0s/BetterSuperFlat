### Usage SinglePlayer:
Select the `Quacon CMP` generation option in the create world screen.


### Usage Multiplayer:

### Does the following:
Overworld -> Blue: Regions, Red: Nether Chunks, Light Gray: Chunk Borders

Nether -> Blue: Regions, Red: Overworld Regions, Light Gray: Chunk Borders
